### 조인 연산
#### 카티전 프로덕트
* 둘 이상의 테이블을 모든 튜플쌍을 출력하는 연산

* 카티전 프러덕트의 SQL 표현 예 : Seclect * <u>from 고객, 주문</u>;
#### Join 연산 
* 두 개 이상의 테이블을 외래키를 중심으로 join(연결)하여 데이터 처리

* 카티전 프로덕트 연산결과에 셀렉션과 프로젝션을 적용하는 것과 비슷
* 일반문법 : select <속성들> from 테이블1, 테이블2 where <조인조건>
and <검색조건>

조인연산 예제(1)

#### 외부 join
 : 연결 조건이 불일치한 튜플 (from절에서 처리한다.)

구매를 하지 않은 고객을 상대로 별도의 마케팅 전략을 할 수 있다. 
-- 도서를 구매하지 않은 고객을 포함하여 고객의 이름과 고객이 주문한 도서의 판매가격을 구하시오.
: select 이름, 판매가격 
from 고객 left outer join 주문 on 고객.고객번호 = 주문.고객번호
"from 절에서 left라는 예약어는 from 절에 나열된 테이블 목록 중에서 
왼쪽에 나열된 테이블의 튜플은 join조건이랑 일치하지 않더라도 나열하라.

### 문자열 함수
#### replace 함수 
* replace(str1, str2, str3) : str 1에서 str2를 str3로 변환하여 반환

#### substring 함수 : 임의의 문자열을 입력 받아서 특정 위치의 문자들을
출력하는 기능, 입력 데이터가 3개 필요
1. 처리 대상 문자열, 2. 시작 위치, 3. 추출 종류 위치
* substring(str, n, m) str에서 n번째부터 m개 문자 반환

####Len 함수 : 글자의 수를 세어주는 함수(단위가 바이트(byte)가 아닌 
문자단위), 
* example : 굿스포츠에서 출판한 도서의 제목과 제목의 글자 수를 확인하시오.


### 날짜 함수

### 정리 

● 대표적인 집계함수로는 SUM, AVG, COUNT, MAX, MIN 등이 있다.

● GROUP BY절은 특정 속성을 기준으로 속성값이 같은 튜플들끼리 그룹핑한다.

● HAVING절에서 그룹핑된 튜플그룹의 선택여부를 결정하기 위한 조건을 설정할 수 있다.

● 조인연산은 둘 이상의 테이블을 외래키를 중심으로 연결하여 데이터를 처리하는 연산이다.

● 조인연산은 카티전프러덕트 연산에 셀렉션과 프로젝션 연산을 적용하는하는 개념이다.

● 외부조인을 통하여 조인조건이 만족되지 않은 튜플도 출력대상으로 선택함으로써 추가적인 정보를 제공할 수 있다.

● SQL 내장함수는 속성이름이나 상수를 입력값으로 받아 기능에 따른 처리를 한다.

● SQL내장함수는 SELECT문의 SELECT절을 작성할 때 많이 활용한다.

● 내장함수의 유형으로는 수학함수, 문자열함수, 날짜함수 등이 있다.
